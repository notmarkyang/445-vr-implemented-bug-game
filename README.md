"A Small Problem" - IAT 445
Core Flow
Bedroom > Tank > Train > Vent > Return to Tank

**Progression Summary**
Bedroom
  - Player examines the tank
  - Finds and collects the bracelet
  - Unlocks shrinking mechanic
Tank
  - Player shrinks down
  - Talks to bug NPC
  - Learns the missing bug went to the train station
Train Scene
  - Player talks to conductor
  - Learns they need to pay a fare
  - Finds ticket in bedroom (while big)
  - Uses ticket machine
  - Talks to conductor again > vent unlocked
Vent Scene
  - Player discovers bug stuck behind vent
  - Needs screwdriver (only obtainable while big)
  - Returns with screwdriver
  - Opens vent
  - Bug is freed and appears outside
  - Bug Returns to Tank
Returns to tank
  - Bug thanks player
  - Final interaction > free roam (end state)
